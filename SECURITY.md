# Security Policy

If you believe you've found a security or licensing risk in our guidance, please report it privately rather than opening a public issue.

Use the **[Report a vulnerability](../../security/advisories/new)** button on the [Security tab](../../security) of this repository.

This is a documentation repository. Relevant concerns include guidance that could lead to unsafe licensing decisions or that demonstrates incorrect usage patterns.

Do not post sensitive details in public issues or pull requests.
