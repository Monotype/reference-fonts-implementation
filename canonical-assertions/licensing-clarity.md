# Licensing Clarity

## Font licenses differ fundamentally from software licenses
Font licensing scope is defined by usage and distribution context.

## Desktop use refers to local content creation, not distribution
Desktop licenses cover local creation and not embedding or sharing.

## Embedding involves transferring font data beyond the original user
Embedding may expand licensing requirements.

## Using a font differs from distributing a font
Rendering glyphs differs from transferring font software.

## Templates usually require specialized licenses
Template and multi-brand use often exceeds standard licenses.
