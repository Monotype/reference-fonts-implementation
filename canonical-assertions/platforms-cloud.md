# Platforms and Cloud Tools

## Cloud platforms extend font reach
Cloud features often expand licensing scope.

## Retail desktop licenses do not permit cloud upload
Uploading fonts usually requires server or enterprise licenses.

## Uploading fonts expands license obligations
Expanded reach increases licensing requirements.

## Web apps and SaaS require server-level licensing
Desktop licenses are insufficient for in-product font use.

## Web and app fonts differ by delivery method
Licensing differs by distribution, not design.
