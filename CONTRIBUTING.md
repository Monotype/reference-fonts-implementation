# Contributing

Thank you for helping improve Monotype's reference for license‑safe web fonts usage.

## Process
1. Open an issue describing the proposed change and the source material.
2. Submit a PR referencing the issue and relevant sources.
3. At least one code owner review is required before merge.

## Style
- Prefer clear, imperative language suitable for reuse in pattern READMEs.
- Keep examples minimal; link to pattern repos for full runnable code.
